### Apakah itu GIT dan GITHUB
Git adalah salah satu software yang bisa melakukan Version
Control System ( sistem yang menyimpan dan mengelola rekaman
perubahan dari source code. Sedangkan
GITHUB adalah website/layanan untuk mengelola project GIT

### Beberapa Istilah dalam GIT
- repo = folder project kita
- commit = rekaman/snapshot dari repo kita
- chechkout = berpindah dari sebuah commit
- hash = penanda unik pada sebuah commit
- branch = cabang bebas dari sebuah commit
- merge = menggabungkan branch
- remote = sumber yang memiliki repo
- clone = mengambil repo dari remote
- push = mengirimkan commit ke repo
- pull = mengambil commit dari repo
- pull request = meminta pemilik repo untuk mengambil perubahan 
                  yang dilakukan
- merge conflict = baris yang sama diubah oleh 2 branch yang berbeda
